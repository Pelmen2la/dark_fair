Для того, чтобы запустить проект надо выполнить следующие шаги:

1. Поставить NodeJS.
2. Поставить MongoDB.
3. Запустить npm install в корне проекта.
4. Запустить npm wp (запускает webpack).
5. Создать файлик с логином\паролем админки по следующему пути: /server/config/admin-auth.json, содержание должно быть примерно таким:

{
    "auth": {
        "login": "root",
        "password": "1234"
    }
}

6. Запустить корневой app.js.
